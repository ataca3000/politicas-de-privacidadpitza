# Politicas de Privacidad - Pitza

Sitio estático con la política de privacidad para el proyecto "Pitza".

Archivos incluidos:
- `index.html` — página principal con la política en español.
- `styles.css` — estilos básicos y responsive.

Despliegue (GitHub Pages):
1. Ve a Settings > Pages en el repositorio.
2. Selecciona la rama `main` y la carpeta `/ (root)`.
3. Guarda y espera unos minutos; el sitio estará disponible en la URL de GitHub Pages.
